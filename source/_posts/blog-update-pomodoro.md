# 🚀 6.20号更新：已在网页中加入番茄钟，随时进行专注学习！

为了在浏览博客和日常学习时拥有一个更加沉浸、高效的无干扰环境，我为博客定制开发并上线了一个**全功能本地番茄工作法（Pomodoro Technique）页面**。

不需要下载繁重的客户端，也不用担心第三方应用收集数据。现在，只需打开博客，即可一键开启属于你的专注时光。

---

## ✨ 核心亮点功能

### 1. ⏱️ 专注与休息自由双自定义
告别传统番茄钟死板的 “25+5” 限制。新版本支持**自由调整专注时长与休息时长**（最高支持 120 分钟专注）：
- **专注时间到**：自动结算并无缝切入休息模式。
- **休息时间到**：再次提醒并切回专注，形成完美的效率正循环。

### 2. 📊 智能数据留存（基于 LocalStorage）
作为静态博客，本功能采用了**隐私优先（Privacy-First）**的设计理念：
- 所有专注历史记录均保存在你当前浏览器的 `LocalStorage` 中。
- **完美结算**：顺利完成目标时长后，系统自动记录 “完成” 状态与时长。
- **中途退场**：若中途因故按下 “重置”，系统会聪明地计算你实际坚持的分钟数，并存下一条 “中断” 记录。你的每一分付出都有迹可循。

### 3. 🔒 输入防呆锁定机制
为了防止在倒计时中途因手误、误触修改输入框数值而导致计时逻辑崩溃，系统在点击 “开始” 后会自动锁定输入框，点击 “暂停” 或 “重置” 后自动安全解锁。

### 4. 🌗 完美适配 Butterfly 主题生态
- **原生暗黑模式**：CSS 样式完全基于 Butterfly 主题的官方原生变量（如 `--card-bg`、`var(--text-main-color)`）开发。无论是白天还是深夜切换到夜间模式，界面均能完美自适应。
- **PJAX 丝滑兼容**：针对 Butterfly 强悍的 PJAX 局部刷新机制进行了底层的 JavaScript 兼容重构。无论你在博客内如何切换页面，番茄钟均能正常加载并在后台防断联运行。

---

## 🛠️ 技术实现简述

1. **结构层 (HTML)**：采用严格左对齐、无前置缩进的纯原生 HTML 编写，完美绕过了 Hexo 默认 Markdown 渲染器（`hexo-renderer-marked`）容易将缩进 HTML 误识别为灰色代码块的语法陷阱。
2. **逻辑层 (JavaScript)**：
   - 引入了**有限状态机 (FSM)** 概念，精确控制 `focus`（专注）与 `break`（休息）状态切换。
   - 使用 `typeof GLOBAL_CONFIG !== 'undefined' && GLOBAL_CONFIG.pjax` 动态监听 `pjax:complete` 事件，确保局部刷新后 DOM 元素的重新绑定。
3. **音频层 (Web Audio API)**：摒弃了传统的外部 `.mp3` 音频文件引入，直接调用浏览器底层的 `AudioContext` 实时合成正弦波（Sine Wave）铃声，实现零网络开销、零延迟的清脆到期提示音。
