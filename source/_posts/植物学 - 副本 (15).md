# 植物组织

## 1. 植物组织概念

- **组织**：在个体发育中，来源相同，形态结构相似，担负一定生理功能的细胞组合或细胞群。
- **个体发育**：多细胞植物的个体发育以受精卵为起点，细胞分裂后分化为不同组织，共同完成植物体的生命活动。
- **器官**：由不同的组织按一定的规律构成了器官。
- **植物的器官**：根、茎、叶、花、果实、种子。

## 2. 植物组织类型

### 2.1 分生组织【重点】

**分生组织**：具有持续分裂能力的细胞群或细胞组合；终身保持旺盛的分裂能力。

#### 按来源分

- **原生分生组织**：是指直接由胚胎保留下来的一群原始细胞，具持久的分裂能力，位于根、茎的最前端，是产生其他组织的 **最初来源**。
- **初生分生组织**：是原分生组织刚衍生的细胞组成，这些细胞在形态上已经出现了最初的分化，但细胞仍具有很强的分裂能力，因此它是一种边分裂、边分化的组织，也可看做是由 **分生组织向成熟组织过渡** 的组织。
- **次生分生组织**：是由成熟组织的细胞，经历生理和形态上的变化，脱离原来的成熟状态（即反分化），重新转变而来的分生组织。

#### 按位置分

1. **顶端分生组织**
   - **分布**：根、茎的顶端。
   - **细胞形态**：细胞几乎为等直径，体积小、核大、胞质浓厚、液泡不明显。
   - **细胞分裂**：横分裂，与根、茎的长轴方向平行；使植物体伸长。
   - **分类**：原生分生组织、初生分生组织。
2. **侧生分生组织**（次生分生组织）
   - **分布**：根、茎等器官中，靠近表面，与组织长轴平行，筒状分布。
   - **细胞形态**：细胞长纺锤形或射线状，液泡发达。
   - **细胞分裂**：分裂方向与器官的长轴垂直，使得根、茎粗壮。
   - **分类**：形成层、木栓形成层。
3. **居间分生组织**（初生分生组织）
   - **分布**：一些植物在发育过程中，在成熟组织中夹着一些未完全分化的分生组织；位于节间的下方，或叶子基部。
   - **特点**：单子叶植物没有侧生分生组织，只有 **顶端分生组织**，但是具有居间分生组织。

### 2.2 成熟组织

**成熟组织**：分生组织衍生的大部分细胞，逐渐丧失分裂能力，进一步生长和分化形成其它各种组织，称为成熟组织，也称永久组织。

| **组织类型** | **分类与特点**                                               |
| ------------ | ------------------------------------------------------------ |
| **保护组织** | 初生保护组织（表皮，活细胞）；次生保护组织（木栓层/周皮）    |
| **机械组织** | 厚壁组织（纤维、石细胞，死细胞）；厚角组织（活细胞）         |
| **输导组织** | 木质部；韧皮部                                               |
| **薄壁组织** | 同化、贮水、储藏、通气、传递                                 |
| **分泌组织** | 内分泌（分泌腔/道/细胞、乳汁管）；外分泌（腺鳞、腺毛、蜜腺、排水器） |

#### 2.2.1 保护组织

分布与功能：覆盖于植物的体表，以减少植物失水，防止病原微生物的侵入，控制与外界的气体交换。

- **初生保护组织（表皮）**：一层或多层不规则扁平细胞平铺而成，紧密镶嵌；含表皮细胞、气孔器、附属物。
- **次生保护组织（周皮）**：裸子植物、双子叶植物加粗生长时，表皮被其替代，由多层细胞组成。

#### 2.2.2 薄壁组织（基本组织）【重点】

- **分布与功能**：植物体各器官中均含有。
- **细胞特点**：细胞壁较薄（仅初生壁），液泡大，排列松散，有细胞间隙。
- **分类**：储水组织、同化组织、储藏组织、通气组织。

#### 2.2.3 机械组织

细胞特点：细胞壁加厚，起支持作用。

- **厚角组织**：角隅处不均匀增厚，**活细胞**，含初生壁，支持力较弱。
- **厚壁组织**：具有均匀增厚的**次生壁**，成熟时为**死细胞**，木质化。
  - **石细胞**：多等直径。
  - **纤维**：细长，两端尖细。分木纤维（木质部）和韧皮纤维（韧皮部）。

#### 2.2.4 输导组织【重点】

功能：长距离输导水分、无机盐及有机物。

- **木质部（输导水/无机盐）**：
  - **导管**：死细胞，具木质化次生壁，末端溶解形成穿孔。
  - **管胞**：死细胞，木质化，两端无穿孔。
- **韧皮部（输导有机物）**：
  - **筛管**：活细胞（无核），末端形成筛板，侧面有筛域。
  - **伴胞**：活细胞，与筛管并列，协助物质运输。
  - **筛胞**：存在于裸子植物，不形成筛板，仅靠筛域相通。

> **导管 VS 管胞**：导管运输效率高；多数裸子植物仅有管胞，绝大多数被子植物均有导管和管胞。

#### 2.2.5 分泌组织

植物体内一些特化结构，可分泌物质并积累于细胞内或排出体外。

## 3. 植物组织系统

- **复合组织**：多种组织按一定规律结合构成的单位。
- **组织系统**：一种或几种组织在结构和功能上组成的单位。维管植物分为三种组织系统（表皮系统、基本组织系统、维管组织系统）。
