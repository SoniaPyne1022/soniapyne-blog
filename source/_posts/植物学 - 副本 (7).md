# 红树植物群落

## 1. 红树植物的生境

- **地质地貌**：与珊瑚礁并称为“陆地建造者”。
- **温度**：向高纬度分布的限制因子。要求最冷月平均气温 > 20℃，且月温差 < 10℃。
- **盐度**：受海水浸渍的盐生沼泽，盐度随距离河口远近及种类而变。
- **潮汐与洋流**：
  - **干湿交替**：每天受潮汐影响形成干湿日变化，而非持续积水。
  - **繁殖与分布**：潮汐流动利于果实与胚轴扩散；洋流属性影响其纬度分布。

## 2. 红树植物形态学适应

### 2.1 根系适应

- **固定与支持**：
  - **支柱根与板状根**：由茎下部伸出拱入土壤，增强抗风浪能力，且具皮孔以利通气。
- **呼吸与通气**：
  - **气生根**：从枝条下垂。
  - **呼吸根**：横走地表，呈膝状或垂直向上。富含**气道**，通过大皮孔疏导空气，满足根部缺氧环境下的呼吸需求。

### 2.2 胎生现象

- **胎生**：种子在母树果实内发芽，发育为**胎生胚轴**。
- **隐胎生**：幼苗在果皮内萌发，果实掉落后涨破果皮，幼苗伸出固着（如桐花树、白骨壤）。
- **适应性**：比重小、易漂浮的胚轴具高抗盐性，漂流数月仍能存活。长花期及果实分批成熟利于定居。

### 2.3 旱生与抗盐结构

面对强风、高热导致的“生理干旱”：

- **叶片旱生结构**：革质或肉质、全缘；厚角质层；气孔深藏；具储水组织；栅栏组织间含石细胞；叶下密被茸毛；叶片与光平行避免直射。
- **高渗透压**：细胞渗透压远高于陆生植物，以应对高盐基质。
- **富含单宁**：抑制细菌，防止动物啃食。
- **盐分代谢**：
  - **拒盐植物**：根系过滤，通过木质部负压分离淡水。
  - **泌盐植物**：通过盐腺将盐分分泌至叶表，随雨水冲洗或叶片脱落移除。

## 3. 红树植物的地理分布

木本植物分为两类：

- **红树植物**：仅在红树林中生长的植物。
- **半红树植物**：既能在红树林中生长，也能在陆地环境生长的植物。

## 4. 生态作用与经济价值

1. **资源丰富**：蕴藏极高的生物多样性。
2. **海岸防护**：护堤固滩、防风消浪、降低盐害。
3. **环境净化**：吸收污染物，减轻海水富营养化，预防赤潮。
4. **科教旅游**：最具特色的湿地生态系统，兼具陆地与海洋生态特性。
